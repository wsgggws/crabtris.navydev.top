# Crabtris

俄罗斯方蟹 = Crab（蟹）+ Tetris（俄罗斯方块）

一个独特的横向俄罗斯方块游戏，以螃蟹为主题。在这个游戏中，方块会从左向右移动，你需要通过上下移动和旋转来适应游戏区域，消除完整的列并获得高分。

## 运行

```bash
gcc -o crabtris crabtris.c
./crabtris  # 注意：不是 carbtris
```

## 效果

![俄罗斯方蟹](./crabtris.png)

## 说明

- 该蟹身长 22 坐标点，宽 10 坐标点。
- 方块向右前进，可上下移动与旋转。
- 没有开始选项。
- 没有暂停选择。
- 结束只能Ctrl+c。
- 能消列，但不计分数，只看能玩多久
- 蟹如人生呀！
